# Make Bilibili Great Again!

## 功能

目前只支持国际版(com.bilibili.app.in)

### 首页

* [X] 干掉竖屏

### 视频播放页

* [X] 干掉播放器内展示的关注弹窗、投票弹窗等（包括UP主弹幕）
* [X] 视频标题左边的“热门”等标签
* [X] 点击评论区文字不会自动弹出回复区

### 设置

* [ ] 在B站设置界面的标题上加上一个MBGA的按钮，点击打开MBGA的设置
* 配置项
  * [ ] 是否要干掉UP主弹幕
